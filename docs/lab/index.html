반응형 UI 프레임 실험실

목표: 배경 타일(반복) + CSS UI 프레임이 어떤 기기 폭에서도 빈칸 없이 자연스럽게 보이는지 확인.

  • 배경 타일 파일명: bg_tile.jpeg
  • 테스트 폭: 360 / 390 / 768 / 1024+ (창 크기 조절로 확인)
  • 이 페이지는 /docs/lab/에서만 실험

패널 레이아웃

모바일에서는 1열, 태블릿/PC에서는 2~3열로 재배치되는지 확인.

공지

공지 문장이 길어져도 버튼/레이아웃이 깨지지 않는지 확인.

일정

타임테이블/운영시간 등 2~3줄 정도를 넣어도 안전한지 확인.

굿즈

굿즈 이름이 길어져도 줄바꿈이 자연스러운지 확인.

럭키드로우

참여 방법/주의사항 텍스트가 많아도 패널이 안정적인지 확인.

오시는 길

지도 버튼이 눌리기 쉬운 크기(터치 영역)인지 확인.

FAQ

Q/A 목록을 넣을 때 스크롤과 간격이 자연스러운지 확인.

체크리스트

디버그

현재 화면 폭을 보기 위한 자리(추후 JS로 표시 가능). 지금은 창 크기 조절로 확인.

Tip : 모바일에서는 주소창/툴바로 높이가 바뀌니, 배경이 끝까지 깔리는지 스크롤/회전으로 꼭 확인.
